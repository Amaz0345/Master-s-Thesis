Data for the scripts can be downloaded from: https://owncloud.gwdg.de/index.php/s/q11668N0KOUsjUK
The first dataset, data_2019, is the version that is compatible with the R scripts finalRcode. 
The dataset couples_2019 includes all data for couples in the 2019-21 DHS, matched according to men's line number, cluster number and household number, and includes instructions for merging the DHS 2019-20 Women's Dataset with the Men's Dataset, and the shdist individual district identifier from the PR (Household) Dataset
